領導者的人脈，伊巴拉這篇的三種網

很多人升到某個位置就卡住了，而且卡得莫名其妙。他明明是全公司做事最紮實的那一個——交給他的案子從不出包，技術細節倒背如流，加班到最晚。可是要再往上一階、真正帶一個部門的時候，公司挑的往往是別人。他心裡多半還嘀咕：那些整天在外面跟人喝咖啡、交際應酬的同事，論本事根本比不上我。

把這個卡關講到最透徹的，是組織行為學者埃米妮亞・伊巴拉與馬克・杭特(Herminia Ibarra & Mark Hunter)，2007 年在《哈佛商業評論》發表的〈How Leaders Create and Use Networks〉(領導者如何經營與運用人脈)。他們追蹤了一批正在往管理層爬的經理人，發現一件事：越是靠「把事情做到最好」爬上來的專才，越把「經營人脈」看成勢利、油滑、浪費時間，於是抗拒它——而這恰好就是他們過不了領導那道關的原因。這裡不談文章裡的細節，只用最白話的方式，帶你抓住我認為最該記住的三種人脈網。

第一種：營運人脈，把手邊的事做好

第一種叫營運人脈，就是你要把眼前這份工作做完做好，非合作不可的那群人。你的組員、隔壁部門的窗口、常配合的供應商、會幫你擋事的行政同事——每天實際把事情推動起來，靠的就是這張網。

這張網有兩個特點：一是對象大多是「被指派」的，你不太能自己選，誰在這個流程裡你就得跟誰合作；二是目標很清楚，就是把當期的任務準時交出去。想像一位很強的工程師，他跟測試、跟 PM、跟採購都很熟，案子跑得順。這張網讓他成為「很會做事的人」，但也僅止於此——它幫你把今天的事做完，但很難幫你看見明天。

第二種：個人人脈，把視野撐開

第二種叫個人人脈，是你圈子以外、跟工作沒有直接業務關係的人：同業聚會認識的前輩、以前的同學、在其他產業做事的朋友、社團裡談得來的人。他們幫不上你這週的專案，卻能給你營運人脈給不了的東西——不同產業的做法、對某個決定的第二種看法，還有關鍵時刻的一句轉介。

舉個台灣常見的例子。一位業務主管想從只管業績，轉去帶整個事業單位，他卡在「不知道別家公司的組織是怎麼長的」。這時候真正幫上忙的，往往是他在某個產業社群認識、在別家公司帶過團隊的朋友，一句「我們當年是這樣分權的」就替他打開一扇窗。個人人脈的價值往往要過一陣子才顯出來——它把你的視野撐開，也常常是通往下一種、也是最難的那種網的入口。

第三種：策略人脈，看見未來、動員資源

第三種叫策略人脈，伊巴拉與杭特說這是最重要、卻最少人真正在經營的一種。它指的是那些能幫你看清未來方向、並在你需要時替你動員資源、跨部門把事情推成的人——高層、其他單位的主管、握有預算或影響力的關鍵角色，他們多半不歸你管，卻決定了你的目標成不成。

策略人脈的精髓，是把一個地方拿到的資訊、支持與資源，調度到另一個地方去達成目標。一位新上任的部門主管，如果只會埋頭把自己部門的事做好(那是營運人脈的老本行)，很快會發現方向根本不是他一個人能定的：預算在財務手上、資源要跟別的單位搶、公司三年後要走哪裡是上面在畫。他得經營的，是能一起把餅畫大、關鍵時刻願意挺他的那群人。這一步最難，因為它跟前半生「靠自己把事做到滿分」的本能是相反的。

從專才到領導，關卡就在後兩種

把三種網放在一起看，伊巴拉與杭特的洞見就浮出來了：專業工作者靠營運人脈就能發光，可是一旦要當領導，光有這張網遠遠不夠。從專才晉升到領導者，最大的關卡就是學會經營個人人脈，尤其是策略人脈——從「一個人把事做到最好」的踏實，走進「透過一群人把方向做成」的模糊地帶。

難就難在，很多技術底的主管把後兩種網看成應酬、看成不務正業，寧可回頭多做一件自己拿手的事。可是領導的工作，本來就是讓一群你管不到的人願意往同一個方向出力，而不只是自己埋頭把事做完。抗拒經營人脈，等於把自己鎖在「很會做事」這一格，永遠上不去。

帶得走的一句自問

把這篇收成一句今天就能用的檢查：攤開你這一週真正花時間互動的人，問自己——「這裡面有幾個，是能幫我看見一年後方向、或在關鍵時刻替我動員資源的人？」如果答案幾乎是零，你的網可能只夠你把眼前的事做好，還撐不起領導。經營人脈聽起來很油，說穿了不過是刻意去認識那些你未來一定用得上、卻還沒認識的人。

伊巴拉與杭特這篇〈How Leaders Create and Use Networks〉原文收在《哈佛商業評論》2007 年 1 月號，想讀完整的分類與研究論證，很值得找 HBR 原文或哈佛商業評論中文版慢慢讀。想接著看領導與策略的其他角度，可以讀我們整理的波特〈策略是什麼？〉。
