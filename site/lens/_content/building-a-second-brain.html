你存的每條筆記，都該先有個買主

佛特在《打造第二大腦》裡描的願景很迷人：你讀到、聽到、想到任何一個讓你心頭一震的東西，都隨手捕捉下來，交給一個外接的數位大腦保管。這樣你那顆會累、會忘、容量又有限的生理大腦，就被空出來做它最該做的事——想，跟創造。他把整套做法收進四個動作，叫 CODE：捕捉、組織、萃取、輸出，從你看到一句好話，一路接到你做出一個成品。

這類教你收集、整理知識的書，我讀過的擺起來也有大半櫃，從《卡片盒筆記》到一拖拉庫把資料夾切得漂漂亮亮的系統。它們有個從不明說、卻人人照單全收的共同假設：你存下來的東西，將來一定用得到。佛特這本，是其中把「輸出」寫得最認真的一本——可連它，也沒在你按下存檔的那一刻，先攔住你問一句：這條，將來到底是誰要用？

所以這篇不打分數，也不談誰該買。我想做的，是把痛點那套看事情的方法，架到這本書上，戳它一個它沒逼你在最前面回答的問題：你正要存的這一條，有沒有一個「要用它的人」在等？痛點對一門生意盯死的第一件事，就是這個——你做出來的東西，外面到底有沒有人要。把同一把尺轉過來對準你的筆記，整座第二大腦的長相，會變得很不一樣。

「先確認有人要，再動手」，筆記也吃這一套

痛點那本書反覆敲的，是一個很基本、卻最多人跳過的動作：動手之前，先確認外面真的有人要。別憑「我覺得應該有市場吧」就把身家押下去，先去看有沒有人為這個痛掏過錢，再決定要不要做。

一條筆記，其實也是你生產出來的一個小東西。你花了一段注意力把它捕捉、分類、收好，等於替它備了一份庫存。接下來的問題，跟做生意一字不差：你這份庫存，有沒有買主？佛特請你「把所有有共鳴的都收下來」，立意很好，可是照單全收的結果，往往是你囤了一整倉庫的料，每一份看起來都有用，後面卻沒有任何一個「你真的要做出來的東西」在等著領走它。

收藏一整櫃，端上桌的永遠是泡麵

【阿凱是那種看到食譜就順手收藏的人。三年下來，他的收藏夾裡躺著上千道菜，分門別類、標好星等，光用滑的都覺得自己快是個廚師了。可是每天晚上他真正煮的，還是同一包泡麵——要從那上千道裡挑一道、再出門把料買齊，光想就累，不如撕開泡麵。】（舉個例子。料囤了一整櫃、端上桌的永遠是最將就那一樣，這種事大概各行各業都有。）

阿凱滑收藏夾滑得很開心，可那上千道菜，沒有一道在等著被端上某一頓他非煮不可的飯。少了那頓飯，收藏夾再厚，也只是滑著爽。一條筆記同理：捕捉它的當下，後面若沒有一個「你真的要端出去的東西」在等，它就只是又一道你永遠不會煮的收藏菜。佛特其實也看見了這個坑，他要你「只收有共鳴的」，又把輸出放在 CODE 的最後當壓軸。可全書的份量還是偏了：最讓人上癮、寫得最誘人的是「收」這個動作，最該被你練熟的「煮」，反而被擺到最後，也最容易被人翻過去。

替每條筆記，先點名一個買主

那買主長什麼樣？佛特自己其實給了答案，藏在他那套 PARA 分類法裡。他要你別照「主題」把資料分門別類，改照「行動」分——第一順位永遠是 Projects，你手上正在進行、有截止感的那幾件事。說白了，他要你把每條筆記，掛到一個正在動的專案底下。痛點把這件事再往前推一格：在你按下存檔之前，先問一句，這條東西，我手上哪一個正在做的事，現在就用得到它？那個「正在做的事」，就是這條筆記的買主。

找得到買主，這條筆記就值得收，而且你會自動知道該把它放哪——擺到那個專案旁邊，要用的時候伸手就拿得到。找不到買主，先別急著存。它也許真的是好東西，可惜此刻沒有任何一件你在做的事用得上它；這種東西先收進來，只會變成你下次翻找時，又一條得被你略過的雜訊。

把你的第二大腦，算成一個你敢看的數字

痛點那本書有個很核心的習慣：凡是飄在感覺裡、講不清楚的東西，一律逼它變成一個你能心算的數字。一個東西到底有沒有人要，別用「應該有吧」打發過去，去數真的有幾個人掏過錢。你的第二大腦，也能這樣量。

量法很簡單。回想過去三個月，你存進去的東西裡，有幾條真的被你拿出來、用進了一個交出去的成品？再對照這三個月你一共存了多少條。那個比例，就是你這座第二大腦真正的命中率。多數人算完會有點難堪：存了好幾百，用上的不到一隻手數得完。這個數字之所以重要，是它把一個你一直假裝看不到的成本，擺到了眼前——你存的每一條，未來的你都得在翻找時，一條一條從它面前滑過去。囤得越多，那座倉庫越難走進去。收集這件事從來不是免費的，你每多收一條沒買主的，就替未來的自己多埋一條要繞過的雜訊。

所以這禮拜，給自己立一條很小的規矩：下次又讀到一段讓你心頭一震、很想趕快收起來的東西，先別急著按存檔。停個三秒，問一句——我手上正在做的哪一件事，現在就等著用它？答得出來，這條就有了買主，好好收，收到那件事旁邊去；答不出來，就讓它過去。

你大概會怕這樣篩會漏掉好東西。真的做上幾個禮拜，多半會碰到一個有點意外的結果：那些被你放掉的「以後也許用得到」，事後幾乎沒有一條讓你真的後悔；而你手上正在做的那件事，因為手邊只剩下用得上的料，反而推得比以前順。某天你打開那個資料夾，裡頭安安靜靜躺著二十幾條筆記，每一條都正貼著你這陣子在忙的事。你伸手抽出一條，三秒後它就上了工。那一刻你會知道，這座第二大腦，總算開工了。
